# 📖 BookShelf.log

읽은 책의 감상문을 BGE-M3 임베딩으로 벡터화하고,  
유사한 감상끼리 연결하는 인터랙티브 독서 지식 그래프입니다.

## 주요 기능

- 알라딘 API로 도서 표지/메타데이터 자동 수집
- 감상문을 BGE-M3 임베딩으로 벡터화
- 코사인 유사도 기반 감상 연결
- D3.js force graph로 독서 세계 시각화

## 기술 스택

| 영역 | 기술 |
| --- | --- |
| Backend | Python, FastAPI |
| Embedding | BGE-M3 (BAAI/bge-m3) |
| Database | SQLite |
| Frontend | React, D3.js |
| external API | 알라딘 Open API |
